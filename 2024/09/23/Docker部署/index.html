1.CentOS安装Docker

首先介绍一下docker技术：

Docker是一个快速交付应用、运行应用的技术，具备下列优势：

  • 可以将程序及其依赖、运行环境一起打包为一个镜像，可以迁移到任意Linux操作系统
  • 运行时利用沙箱机制形成隔离容器，各个应用互不干扰
  • 启动、移除都可以通过一行命令完成，方便快捷

为了解决依赖的兼容问题的，采用了两个手段：

  • 将应用的Libs（函数库）、Deps（依赖）、配置与应用一起打包
  • 将每个应用放到一个隔离容器去运行，避免互相干扰

Docker 分为 CE 和 EE 两大版本。CE 即社区版（免费，支持周期 7 个月），EE 即企业版，强调安全，付费使用，支持周期 24 个月。Docker CE 分为 stable test 和 nightly 三个更新频道，这里主要介绍 Docker CE 在 CentOS上的安装。

2.CentOS7安装DockerCompose

Docker Compose 是 Docker 的一个子项目，可以帮助用户定义和运行多容器 Docker 应用程序。 可以更轻松地管理多个容器的应用程序，例如在本地或云平台上构建和部署微服务应用程序。

相当于是一种可以让您用一个文件来管理多个容器的工具，基于Compose文件帮我们快速的部署分布式应用，而无需手动一个个创建和运行容器。
