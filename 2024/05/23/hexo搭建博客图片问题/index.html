hexo搭建博客图片问题

在博客搭建完成之后，要显示图片，把图片直接用相对路径或绝对路径引用之后发现无法显示

那么我们就来看一下如何操作：

1.要添加图片，先在hexo博客仓库通过 npm install 一个 hexo-asset-image 的依赖

如下：

PS：由于我先前已经下载了这个依赖，所以结果显示为更新操作

2.然后把_config.yml中的post_asset_folder设为true，这个配置的意思是每次new 一个博客，会增加一个和博客同名的文件夹。

比如我将post_asset_folder设为true之后hexo new “hexo搭建博客图片问题”之后新建了一篇文章，就会同时在hexo/source/_post文件夹下生成一个.md文件和一个同名文件夹，文件夹存放文章中的图片

如下：

3.把图片放到.md文件的同名文件夹”hexo搭建博客图片问题”里，最后在.md文件中引用图片

4.保存文件的修改之后，在命令行输入hexo c(删除静态文件命令),接着输入hexo g(生成静态文件命令),通过hexo d部署后在浏览器输入域名后查看相关文章，显示图片成功！！！
